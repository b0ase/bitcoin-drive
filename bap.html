üì¶ The .nft Container Format

The .nft (Network File Type) is Bitcoin Drive's revolutionary container format designed specifically for BitcoinSV blockchain storage. Like .zip or .tar formats, but optimized for on-chain data.

Core Innovation

The .nft format acts as a universal container that can encapsulate any file type while adding blockchain-specific metadata, encryption layers, and ownership attestation through integration with existing BSV protocols.

üîè Bitcoin Attestation Protocol (BAP)

Bitcoin Drive leverages the Bitcoin Attestation Protocol to create verifiable chains of trust for stored files. BAP provides cryptographic proof of ownership, identity verification, and attribute attestation.

What is BAP?

BAP is a protocol for creating attestations and identity verification on the Bitcoin blockchain. It allows for:

  • Identity Creation Generate unique identity keys linked to Bitcoin addresses without exposing private keys
  • Attribute Attestation Third parties can verify and cryptographically sign identity attributes
  • Chain of Trust Build verifiable trust relationships for data without publishing the actual data
  • Key Rotation Update signing keys without invalidating existing attestations

BAP Integration in Bitcoin Drive

Every .nft file uploaded to Bitcoin Drive can be attested using BAP, creating an immutable record of:

Ownership Proof

Cryptographic attestation linking files to verified identities

Access Control

Define who can decrypt and access files based on attested attributes

Audit Trail

Complete history of file creation, modification, and access

üîß Technical Architecture

Storage Flow

1. File Upload ‚Üí 2. NFT Container Creation ‚Üí 3. BAP Attestation ‚Üí 4. BSV Transaction ‚Üí 5. On-Chain Storage

Supported BSV Protocols

  • B:// Protocol Binary data storage for large files
  • C:// Protocol Content-addressed storage with built-in deduplication
  • D:// Protocol Dynamic state management for mutable data
  • BCAT Protocol Concatenated data for files exceeding single transaction limits
  • MAP Protocol Magic Attribute Protocol for metadata and indexing
  • BAP Protocol Bitcoin Attestation Protocol for identity and trust
